# Poll App

A modern survey and voting application built with Angular.

Users can create surveys, add multiple questions and answers, vote in polls, and view live results.

## Features

- Create surveys
- Add multiple questions
- Add multiple answer options
- Single choice voting
- Multiple choice voting
- Live survey results
- Running & past surveys
- Category filter
- Delete surveys
- Local storage persistence
- Responsive UI design

## Technologies

- Angular 21
- TypeScript
- SCSS
- HTML
